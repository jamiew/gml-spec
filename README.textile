h1. GML Spec

_"The new digital standard for tomorrow's vandals."_

This is the reference Graffiti Markup Language (GML) file format specification.
GML is an XML-based file format specifically designed for archiving graffiti tags.

The spec is available online here:
"http://fffff.at/gml":http://fffff.at/gml
